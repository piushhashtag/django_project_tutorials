Componenet Required:-

Sr No. Componenet Quantity
1. IC CD4017 1
2. LED 6
3. Touch Plate 1
4. Supply(5V) 1

Description:-

When we touch the plate attached to pin number 14 of CD4017, it receive the pulse which is like fast triggering.
LED attached to the output pin of CD4017 start glowing rapidly in sequence.
When we remove the finger from touch plate, it is totally unpredictable which led will remain ON.
This unpredictability makes this project useful in generating random no. which can be used in games.
